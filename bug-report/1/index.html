Bug Report - BUG-101

Не работает кнопка регистрации.

ID
BUG-101
Summary
Кнопка регистрации не реагирует на нажатие.
Description
На странице регистрации кнопка "Зарегистрироваться" визуально доступна, но при клике не происходит отправка формы и не выполняется запрос на сервер.
Steps to Reproduce
1. Открыть страницу регистрации
2. Заполнить все обязательные поля валидными данными
3. Нажать кнопку "Зарегистрироваться"
Expected Result
Форма отправляется, пользователь успешно регистрируется или отображается сообщение об ошибке.
Actual Result
Ничего не происходит, запрос в сеть отсутствует.
Severity / Priority
Severity: Blocker Priority: High
Environment
Windows 11 - Chrome 144 - Web App v0.7559.109
