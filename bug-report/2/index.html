Bug Report - BUG-102

Стили не применяются в медленной сети.

ID
BUG-102
Summary
CSS-стили не загружаются при медленном интернет-соединении.
Description
При низкой скорости сети или нестабильном соединении CSS-файлы не успевают загрузиться. В результате страница отображается без стилей (unstyled content) и не восстанавливается даже после полной загрузки.
Steps to Reproduce
1. Включить ограничение сети (Slow 3G) в DevTools
2. Обновить страницу приложения
3. Дождаться полной загрузки
Expected Result
Стили должны примениться после завершения загрузки ресурсов.
Actual Result
Страница остаётся без CSS-оформления.
Severity / Priority
Severity: Major Priority: Medium
Environment
Android 15 - Chrome Mobile - Web App v0.7559.109
